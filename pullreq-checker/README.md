## pullreq-checker tool
Check Pull/Reqs whether or not they are mergeable

#### Make zip file(distribution)
    $ mvn clean package
     [output] target/pullreq-checker-1.0.0-release.zip

#### Deploy zip file
Deploy zip file and unzip it.

#### Setup
Modify application.properties depending on your environment.

#### Run
    $ sh prchecker.sh
